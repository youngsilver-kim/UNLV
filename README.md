# UNLV AI Winter Exchange (Jan–Feb 2026)

This repository documents key concepts, technical insights, and structured knowledge gained during a short-term Artificial Intelligence exchange program at the University of Nevada, Las Vegas (UNLV).

The purpose of this repository is not just to record the experience, but to transform learning outcomes into reusable technical assets based on mathematical understanding and practical implementation.

---

## Program Information
- Institution: University of Nevada, Las Vegas (UNLV)
- Duration: January 2026 – February 2026
- Field: Artificial Intelligence (Machine Learning, Deep Learning, Data Analysis)

---

## What I Learned

### Machine Learning
- Supervised Learning: Linear Regression, Logistic Regression
- Model Evaluation: MSE, Accuracy, Bias-Variance Tradeoff
- Overfitting and Regularization concepts

### Deep Learning
- Neural Network fundamentals (forward propagation, backpropagation)
- Loss functions and optimization (Gradient Descent)
- Basic architecture understanding

### Mathematical Foundations
- Linear Algebra: vectors, matrices, eigenvalues
- Probability & Statistics: distributions, expectation, variance
- Mathematical reasoning behind model behavior

### Practical Skills
- Python-based implementation of ML models
- Training and evaluation pipeline
- Understanding the gap between theory and real-world performance

---


## Key Takeaways

- AI models are only as good as the assumptions behind them.
- Strong mathematical foundations improve both performance and interpretability.
- Practical implementation exposes limitations that are not visible in theory alone.
- Structuring knowledge is essential for long-term retention and application.
